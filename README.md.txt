# Employee Management System

## 📌 Project Overview
Employee Management System is a Java Web Application developed using JSP, Servlets, JDBC, JPA, and MySQL.  
This application helps manage employee records, including adding, updating, deleting, and viewing employee details efficiently.

---

## 🚀 Technologies Used
- Java
- JSP
- Servlets
- JDBC
- JPA
- MySQL
- Apache Tomcat

---

## 💻 Features
- Add Employee
- Update Employee Details
- Delete Employee
- View Employee Records
- Database Integration using JDBC & JPA
- User-friendly Interface

---

## 🛠️ How to Run the Project
1. Clone the repository
2. Import project into Eclipse as Dynamic Web Project / Maven Project
3. Configure MySQL database
4. Update database connection in code / configuration files
5. Run the project on Apache Tomcat Server
